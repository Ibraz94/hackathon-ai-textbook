# Hackathon AI Textbook

This repository contains the source code and content for the Physical AI & Humanoid Robotics Textbook.

## Development

To run the Docusaurus site locally:

```bash
cd docusaurus-site
npm install
npm start
```

## Deployment

### CDN Strategy

The Docusaurus site is designed to be deployed to GitHub Pages, which provides built-in CDN capabilities for efficient content delivery.
